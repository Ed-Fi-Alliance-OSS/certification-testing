# SIS Artifact

The use of SIS certification scenarios artifact is intended to be used by `Ed-Fi Admin App`, `Ed-Fi Representatives`, and `Ed-Fi Vendors`.

## What is included

- `SIS/` — Bruno collection with all certification scenarios.
- `package.json` and `package-lock.json` — dependency manifest.
- `scripts/`, `schemas/`, `templates/` — supporting runtime files.
- `README-User-Manual.md` — Bruno UI setup and execution guide.
- `README-SIS-Artifact-How-to-use-in-Bruno-CLI.md` — Bruno CLI setup and execution guide.
- `README-SIS-Artifact-How-to-use-in-AdminApp.md` — Admin App setup and execution guide.
